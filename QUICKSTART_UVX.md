# 🚀 百度地图 MCP Server Enhanced - UVX 快速启动指南

## 📦 使用 UVX 一键启动（推荐）

### 快速开始

```bash
uvx bach-baidu-maps
```

### 在 Cursor / Cherry Studio 中配置

在 MCP 配置文件中添加：

```json
{
  "mcpServers": {
    "baidu-maps-enhanced": {
      "command": "uvx",
      "args": ["bach-baidu-maps"],
      "env": {
        "BAIDU_MAPS_API_KEY": "<YOUR_API_KEY>"
      }
    }
  }
}
```

**获取百度地图 API Key：**
前往 https://lbsyun.baidu.com/apiconsole/key 申请

## 🔧 传统安装方式

### 使用 pip 安装

```bash
pip install bach-baidu-maps
```

### 使用 pipx 安装

```bash
pipx install bach-baidu-maps
```

### 启动服务

```bash
export BAIDU_MAPS_API_KEY="<YOUR_API_KEY>"
bach-baidu-maps
```

## 🌟 功能特性

### 1. 地理编码服务

将地址解析为对应的位置坐标

```
查询"北京市海淀区上地十街十号"的坐标
```

### 2. 逆地理编码服务

根据纬经度坐标获取对应位置的地址描述

```
查询坐标 (39.9042, 116.4074) 的地址信息
```

### 3. 地点检索服务

检索城市内或周边的地点信息

```
在北京市查找"咖啡馆"
查找天安门周边1000米范围内的"餐厅"
```

### 4. 地点详情服务

获取指定 POI 的详情信息

```
查询POI详情：uid=xxxxx
```

### 5. 路线规划服务

支持驾车、骑行、步行和公交路线规划

```
规划从"天安门"到"故宫"的步行路线
规划驾车路线：起点(39.9042, 116.4074)，终点(39.9163, 116.3972)
```

### 6. 批量算路服务

批量计算多个起终点之间的路线规划距离和时间

```
批量算路：起点[(39.9042, 116.4074)]，终点[(39.9163, 116.3972), (39.9088, 116.3974)]
```

### 7. 天气查询服务

查询实时天气信息及未来 5 天天气预报

```
查询北京的天气
查询坐标 (39.9042, 116.4074) 的天气
```

### 8. IP 定位服务

通过 IP 获取位置信息

```
查询当前IP位置
查询IP地址：8.8.8.8
```

### 9. 实时路况服务

查询实时交通拥堵情况

```
查询"朝阳路南向北"在北京的路况
查询圆形区域内的路况：中心点(39.9042, 116.4074)，半径500米
```

### 10. POI 智能提取

从文本中智能提取 POI 信息

```
提取文本中的地点信息："明天去三里屯的星巴克见面"
```

## 🔐 注意事项

1. **API Key 安全**: 请妥善保管您的百度地图 API Key，不要在公开代码中暴露
2. **配额限制**: 百度地图 API 有请求配额限制，请注意使用频率
3. **坐标系统**: 百度地图使用 BD-09 坐标系统
4. **高级功能**: POI 智能提取功能需要申请高级权限
